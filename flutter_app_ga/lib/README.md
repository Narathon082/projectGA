Refactoring Plan: แตกไฟล์ graph_page.dart
แผนนี้จะทำการแยกส่วนประกอบ (Widgets) และโค้ดตรรกะต่างๆ ออกเป็นไฟล์ย่อยๆ เพื่อให้เป็นระเบียบและดูแลรักษาได้ง่ายขึ้น

โครงสร้างโฟลเดอร์ใหม่ที่จะสร้างขึ้น
ผมจะสร้างโฟลเดอร์ใหม่ภายใน lib/ เพื่อแยกหมวดหมู่ไฟล์ให้ชัดเจน:

1. lib/models/
energy_data.dart: ย้ายคลาส EnergyData ออกมา
2. lib/constants/
app_colors.dart: ย้ายคลาสสี _C ออกมา เพื่อให้ทุกไฟล์สามารถเรียกใช้ค่าสีเดียวกันได้
3. lib/widgets/ (แยกชิ้นส่วน UI)
top_bar.dart: ส่วนหัวของแอป (โลโก้ + ปุ่มเปลี่ยน Dark Mode)
power_card.dart: การ์ดแสดงผลค่าไฟ Max/Current
trend_chart.dart: กราฟแสดงผล (LineChart) และปุ่มสลับ Day/Week
activity_logs.dart: ส่วนแสดงประวัติรายชั่วโมง/รายวันด้านล่างกราฟ
history_list.dart: หน้ารายการประวัติย้อนหลัง (List ของวันที่ทั้งหมด)
bottom_nav.dart: แถบเมนูด้านล่างสุด (หน้าหลัก / ประวัติ)
4. lib/pages/
graph_page.dart: ไฟล์หลักจะถูกย้ายมาที่นี่ โดยจะทำหน้าที่แค่เรียกใช้ Widget จากไฟล์ย่อยๆ ด้านบนมาประกอบกัน และดึงข้อมูลจาก Firebase (StreamBuilder) ทำให้ไฟล์นี้จะเหลือความยาวแค่ประมาณ 150-200 บรรทัดเท่านั้น
⚠️ สิ่งที่ต้องพิจารณา (User Review Required)
IMPORTANT

การแตกไฟล์จะไม่มีผลต่อการทำงานของแอปพลิเคชัน (ฟีเจอร์ยังคงเดิมทุกอย่าง) แต่จะเป็นการ จัดระเบียบโค้ดใหม่ทั้งหมด เพื่อให้อ่านง่ายขึ้น
