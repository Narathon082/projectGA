# Energy Monitor (Watt Monitor) ⚡

แอปพลิเคชัน Flutter สำหรับการมอนิเตอร์และติดตามการใช้พลังงานไฟฟ้า (Watt และ Amp) แบบเรียลไทม์ โดยดึงข้อมูลที่ถูกส่งมาจากอุปกรณ์ IoT (เช่น ESP32 หรือ Arduino) ผ่าน Firebase Realtime Database 

---

## 🌟 ฟีเจอร์หลัก (Features)

1. **Daily Dashboard (โหมดดูรายวัน)**
   - แสดงกราฟแนวโน้มการใช้พลังงานไฟฟ้า (Watt) เป็นรายชั่วโมงตั้งแต่ 0:00 - 23:00 น.
   - มีการ์ดแสดงค่าไฟปัจจุบัน/ค่าสูงสุดของวัน พร้อมวันที่ระบุชัดเจน
   - ส่วน Activity Logs แสดงรายละเอียดข้อมูล Watt และ Amp ที่เกิดขึ้นในแต่ละชั่วโมง

2. **Weekly Summary (โหมดดูรายสัปดาห์)**
   - สรุปแนวโน้มการใช้พลังงานสูงสุด (Max Power) ย้อนหลัง 7 วันล่าสุด
   - ช่วยให้สามารถวิเคราะห์และเปรียบเทียบความแตกต่างของการใช้ไฟในแต่ละวันได้ง่ายขึ้น

3. **History Logs (หน้าประวัติย้อนหลัง)**
   - ดูข้อมูลย้อนหลังได้ทุกวันที่มีการบันทึกไว้ในระบบ
   - แสดงรายการวันที่ทั้งหมด (รูปแบบ YYYY-MM-DD) เมื่อกดเข้าไปจะแสดงกราฟของวันนั้นๆ โดยเฉพาะ

4. **Dark Mode / Light Mode**
   - รองรับการสลับโหมดสีสว่างและสีมืด (สลับได้ที่ไอคอนมุมขวาบน)
   - สีของระบบถูกออกแบบมาให้อ่านง่าย สบายตา ทั้งตัวหนังสือและกราฟ

---

## 🛠️ โครงสร้างฐานข้อมูล Firebase (Database Schema)

แอปพลิเคชันนี้ทำงานร่วมกับ **Firebase Realtime Database** โดยคาดหวังให้ฝั่ง Hardware (IoT) ส่งข้อมูลเข้ามาตามโครงสร้างแบบนี้:

```json
"history": {
  "2026-06-01": {
    "8": { "watt": 80.5, "amp": 8.1 },
    "9": { "watt": 105.0, "amp": 10.2 }
  },
  "2026-06-02": {
    "10": { "watt": 120.0, "amp": 12.0 },
    "11": { "watt": 90.0, "amp": 9.2 }
  }
}
```

**คำอธิบาย Path:**
- `history` -> โหนดหลักของฐานข้อมูล
- `YYYY-MM-DD` -> วันที่ปัจจุบัน (เช่น `2026-06-02`)
- `HH` -> เวลา (ชั่วโมง) ตั้งแต่ `0` ถึง `23` (ไม่ต้องมีเลข 0 นำหน้า)
- `watt` และ `amp` -> ค่าพลังงานที่อ่านได้จากเซ็นเซอร์ (รองรับทั้งจำนวนเต็มและทศนิยม)

## 📱 วิธีการใช้งานแอปพลิเคชัน

1. **การดูข้อมูลของวันนี้ (หน้าหลัก)**
   - ทันทีที่เปิดแอปพลิเคชันขึ้นมา ระบบจะแสดงผลการใช้ไฟของ "วันนี้" โดยอัตโนมัติ
   - สามารถเลื่อนดูกราฟ (ใช้มือแตะที่จุดบนกราฟเพื่อดูค่า Tooltip) และเลื่อนดู Log รายชั่วโมงด้านล่างได้

2. **การสลับดู 7 วันล่าสุด (Day / Week)**
   - ในหัวข้อ Consumption Trend จะมีปุ่ม `Day` และ `Week`
   - หากเลือก `Week` กราฟและ Log ด้านล่างจะถูกเปลี่ยนเป็นการแสดงผลค่า "สูงสุด (Max)" ของ 7 วันล่าสุดที่มีข้อมูล

3. **การดูประวัติย้อนหลัง**
   - แตะที่แท็บ **"ประวัติ"** บริเวณเมนูด้านล่างสุด (Bottom Navigation)
   - จะพบกับรายชื่อวันที่ทั้งหมดที่มีข้อมูลใน Firebase
   - แตะที่วันที่ต้องการดู ระบบจะพากลับมาที่หน้า Dashboard พร้อมโหลดข้อมูลกราฟของวันนั้นๆ ทันที

4. **การเปลี่ยนสีหน้าจอ**
   - แตะที่ไอคอนรูป พระอาทิตย์ ☀️ / พระจันทร์ 🌙 มุมขวาบนของแอปพลิเคชันเพื่อสลับโหมดสี

## 🚀 การติดตั้งและรันโปรเจกต์ (สำหรับนักพัฒนา)

1. ตรวจสอบให้แน่ใจว่าได้ติดตั้ง [Flutter SDK](https://docs.flutter.dev/get-started/install) แล้ว
2. คลอน (Clone) หรือดาวน์โหลดโปรเจกต์นี้
3. เปิด Terminal และรันคำสั่งเพื่อดาวน์โหลดแพ็กเกจที่จำเป็น:
   ```bash
   flutter clean
   flutter pub get
   ```
4. รันโปรเจกต์ (สามารถรันบน Chrome, Android, iOS หรือ Windows Desktop ได้):
   ```bash
   flutter run
   ```

*หมายเหตุ: โปรเจกต์นี้มีการเชื่อมต่อกับ Firebase ผ่านไฟล์ `firebase_options.dart` เรียบร้อยแล้ว หากมีการเปลี่ยน Project Firebase อย่าลืมตั้งค่าและคอนฟิกใหม่ผ่าน `flutterfire configure`*
