# Energy Monitor (Watt Monitor) ⚡

แอปพลิเคชัน Flutter สำหรับการมอนิเตอร์และติดตามการใช้พลังงานไฟฟ้า (กำลังไฟ, แรงดันไฟ, และกระแสไฟ) ทั้งฝั่งขาเข้าและขาออกแบบเรียลไทม์ โดยดึงข้อมูลที่ถูกส่งมาจากอุปกรณ์ IoT (เช่น ESP32 หรือ Arduino) ผ่าน Firebase Realtime Database พร้อมระบบกราฟวิเคราะห์อัจฉริยะ

---

## 🌟 ฟีเจอร์หลัก (Features)

1. **Dashboard & Live Power Flow (หน้าหลักและแผงพลังงานเรียลไทม์)**
   - แสดงการ์ดสรุปค่ากำลังไฟฟ้า (วัตต์) และกระแสไฟฟ้า (แอมป์) ล่าสุด
   - แสดงตารางเปรียบเทียบพลังงาน **ขาเข้า (Solar Input)** และ **ขาออก (Battery Output)** แบบเคียงข้างกัน (Side-by-side Grid Card) เพื่อติดตามแรงดันไฟ (V), กระแสไฟ (A), และกำลังไฟวัตต์ (W) ณ ปัจจุบัน
   - ส่วน Activity Logs แสดงรายละเอียดข้อมูล Watt และ Amp ย้อนหลังในแต่ละชั่วโมงหรือแต่ละวัน

2. **Smart Charts (กราฟวิเคราะห์พลังงาน 3 โหมด)**
   - **Realtime Mode (กราฟเรียลไทม์):** กราฟเส้นแบบเคลื่อนไหวและเลื่อนไหล (Scrolling Chart) พล็อตค่ากำลังไฟฟ้าวัตต์เทียบกับวินาที/เวลาปัจจุบัน (Watt/Time) อัปเดตข้อมูลสดใหม่ทุก 2 วินาที
   - **Daily Mode (กราฟรายวัน):** พล็อตกราฟแนวโน้มกำลังวัตต์ย้อนหลังรายชั่วโมง (0:00 - 23:00 น.) ของวันที่เลือก เพื่อใช้วิเคราะห์ช่วงเวลาการผลิตพลังงาน
   - **Weekly Mode (กราฟรายสัปดาห์):** พล็อตกราฟสรุปพลังงานสูงสุด (Max Power) ย้อนหลัง 7 วันล่าสุดของระบบ

3. **Detailed In/Out Screen (หน้าจอวิเคราะห์พลังงาน ขาเข้า-ขาออก ละเอียด)**
   - แสดงผลข้อมูลประสิทธิภาพการชาร์จของระบบ **(Charging Efficiency %)** ด้วย Gauge Chart วงกลมแบบเรียลไทม์
   - คำนวณและแสดงค่ากำลังไฟฟ้าที่สูญเสียในระบบ (Power Loss) เป็นวัตต์
   - วิเคราะห์ระบุสถานะและโหมดการชาร์จแบตเตอรี่แบบอัตโนมัติ (เช่น Bulk - ประสิทธิภาพสูง, Absorption - ชาร์จคงที่, Float - ชาร์จประคองประจุ)

4. **History Logs (หน้าประวัติย้อนหลัง)**
   - ดูข้อมูลย้อนหลังแยกตามรายวันที่ถูกบันทึกไว้ในระบบ Firebase
   - ค้นหาและสืบค้นคีย์วันที่ (รูปแบบ YYYY-MM-DD) เมื่อกดคลิกจะทำการโหลดประวัติและเรนเดอร์กราฟรายวันของวันนั้นๆ ให้ทันที

5. **Dark Mode / Light Mode**
   - รองรับการสลับโหมดสีสว่างและสีมืด (สลับโหมดได้ด้วยปุ่มไอคอนที่มุมขวาบนของทุกหน้าจอ)
   - ออกแบบชุดคู่สีตามมาตรฐาน UI ดีไซน์ทันสมัย อ่านง่าย สบายตา

## 🛠️ โครงสร้างฐานข้อมูล Firebase (Database Schema)

แอปพลิเคชันเชื่อมโยงกับ **Firebase Realtime Database** โดยรับและแลกเปลี่ยนข้อมูลกับบอร์ดฮาร์ดแวร์ตามโครงสร้าง JSON Tree ดังนี้:

```json
{
  "realtime": {
    "vin": 18.5,
    "iin": 1.2,
    "pin": 22.2,
    "vout": 13.8,
    "iout": 1.5,
    "pout": 20.7
  },
  "history": {
    "2026-06-11": {
      "20": { "watt": 8.86, "amp": 0.70 },
      "21": { "watt": 6.38, "amp": 0.51 }
    },
    "2026-06-12": {
      "10": { "watt": 12.50, "amp": 1.05 }
    }
  }
}
```

### คำอธิบายโครงสร้างโหนดข้อมูล:

#### 1. โหนดย่อย `realtime` (ค่าสดเรียลไทม์ปัจจุบัน)
*ใช้สำหรับแสดงผลหน้าแดชบอร์ดและหน้าวิเคราะห์ In/Out หน้าที่ 3*
- `vin` : แรงดันไฟฟ้าขาเข้าจากแผงโซลาร์เซลล์ (Solar Input Voltage - V)
- `iin` : กระแสไฟฟ้าขาเข้าจากแผงโซลาร์เซลล์ (Solar Input Current - A)
- `pin` : กำลังไฟฟ้าขาเข้าจากแผงโซลาร์เซลล์ (Solar Input Power - W)
- `vout` : แรงดันไฟฟ้าชาร์จเข้าแบตเตอรี่ (Battery Charging Voltage - V)
- `iout` : กระแสไฟฟ้าชาร์จเข้าแบตเตอรี่ (Battery Charging Current - A)
- `pout` : กำลังไฟฟ้าชาร์จเข้าแบตเตอรี่ (Battery Charging Power - W)

#### 2. โหนดย่อย `history` (ประวัติเฉลี่ยสะสมรายชั่วโมง)
*ใช้สำหรับแสดงผลหน้าประวัติย้อนหลังและกราฟช่วงรายวัน/สัปดาห์*
- `YYYY-MM-DD` : คีย์วันที่อ้างอิงข้อมูล (เช่น `2026-06-11`)
- `HH` : ชั่วโมงที่จัดเก็บบันทึก (0-23 น.) โดยไม่มีเลข 0 นำหน้า เพื่อความกระชับ
- `watt` : ค่ากำลังไฟฟ้าเฉลี่ยสะสมในชั่วโมงนั้น (ทศนิยม 2 ตำแหน่ง - W)
- `amp` : ค่ากระแสไฟฟ้าเฉลี่ยสะสมในชั่วโมงนั้น (ทศนิยม 2 ตำแหน่ง - A)

*หมายเหตุ: ในแอปพลิเคชันมีระบบจำลองข้อมูลค่าไฟเรียลไทม์อัตโนมัติ (Simulation Engine) เป็นระบบสำรอง (Fallback) หากโหนด `realtime` บนฐานข้อมูลหลักไม่มีค่าหรือออฟไลน์อยู่ เพื่อความต่อเนื่องในการทดสอบระบบ*

## 📱 วิธีการใช้งานแอปพลิเคชัน

1. **การมอนิเตอร์และดูสถานะหลัก**
   - แตะเลือกแท็บ **"หน้าหลัก"** ด้านล่างเพื่อตรวจสอบกำลังวัตต์ปัจจุบัน และแผงเปรียบเทียบกระแส/แรงดัน ขาเข้า-ขาออก แบบเรียลไทม์
   - ในหัวข้อกราฟ สามารถกดสลับดูได้ 3 โหมด คือ `Realtime`, `Day`, และ `Week` เพื่อดูการสลับกราฟพล็อต
   - สำหรับโหมดเรียลไทม์ กราฟจะทำการขยับตัวเองเลื่อนไปทางขวาตามเวลาจริง

2. **การดูและวิเคราะห์ประสิทธิภาพพลังงานอย่างละเอียด**
   - แตะเลือกแท็บ **"ขาเข้า/ขาออก"** (ขวาสุดบนแถบเมนูด้านล่าง)
   - หน้าจอจะแสดงเกจวัดเปอร์เซ็นต์ประสิทธิภาพการชาร์จแบตเตอรี่ของระบบ MPPT, อัตราการสูญเสียกำลังวัตต์ในรูปความร้อน (Loss), โหมดการชาร์จปัจจุบัน รวมถึงแยกตารางแรงดันและกระแสฝั่งแผงและฝั่งแบตเตอรี่อย่างละเอียด

3. **การดูประวัติย้อนหลัง**
   - แตะเลือกแท็บ **"ประวัติ"** เมนูด้านล่างสุด
   - รายชื่อวันที่ทั้งหมดจะแสดงขึ้นมา ให้แตะวันที่ต้องการศึกษา ระบบจะนำคุณกลับไปยังหน้าแดชบอร์ดพร้อมแสดงผลข้อมูลและกราฟประวัติย้อนหลังของวันนั้นๆ ทันที

## 🚀 การติดตั้งและรันโปรเจกต์ (สำหรับนักพัฒนา)

1. ตรวจสอบให้แน่ใจว่าเครื่องคอมพิวเตอร์ได้ติดตั้ง [Flutter SDK](https://docs.flutter.dev/get-started/install) แล้ว
2. เปิด Terminal ในตำแหน่งโฟลเดอร์โปรเจกต์ `flutter_app_ga`
3. รันคำสั่งเคลียร์และดึงไลบรารี:
   ```bash
   flutter clean
   flutter pub get
   ```
4. รันโปรเจกต์ไปยังอุปกรณ์ที่เชื่อมต่ออยู่:
   ```bash
   flutter run
   ```
5. หรือบิลด์ไฟล์ติดตั้งประเภท APK สำหรับอุปกรณ์ Android:
   ```bash
   flutter build apk
   ```
   *ไฟล์ APK ที่สร้างเสร็จแล้วจะอยู่ที่: `build/app/outputs/flutter-apk/app-release.apk`*
